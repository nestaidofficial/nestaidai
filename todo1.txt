PROCEDURE (how to request indexing for one URL)
-----------------------------------------------
1. Click the top search bar ("Inspect any URL in nestaid.us").
2. Type the full https URL, press Return.
3. Wait ~5s for "URL is not on Google" to load.
4. Click "REQUEST INDEXING" (right side, under the result card).
5. It runs a live test ("Testing if live URL can be indexed", ~10-60s).
6. Confirm the green "Indexing requested" dialog appears, click "Dismiss".
7. Repeat for next URL until daily quota is hit (Google will show a quota
   message if exceeded — stop there and resume next day).
